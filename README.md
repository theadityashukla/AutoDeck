# AutoDeck

AutoDeck is an intelligent presentation generator that creates PowerPoint presentations using RAG (Retrieval Augmented Generation) technology. The system processes scientific documents and generates structured, coherent presentations tailored to your specific needs.

## Work in Progress

This is a personal project currently under active development. The goal is to streamline the process of creating informative presentations from scientific documents while maintaining accuracy and relevance.

## Features Planned

- Structured input system for presentation requirements
- Scientific document parsing using GROBID
- Smart content retrieval using Vector DB
- LLM-powered content generation and organization
- PowerPoint generation with proper structure
- Support for templates and theming
- Citation management
